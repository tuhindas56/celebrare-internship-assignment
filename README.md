# celebrare-internship-assignment

Internship assignment made for Celebrare

## Features

- Responsive grid of photos
- Real-time search filtering
- Ability to mark photos as favourites and persist favourites state across refreshes

## Tech stack

React,
TailwindCSS,
Vite

## Setup

### Development

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

### Production

Build the project

```bash
npm run build
```
